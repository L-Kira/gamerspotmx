B) CSS Files and Structure - top

I'm using five CSS files to style this template

  1. style.css = The main css file that is used to style this template
  2. color.css = This CSS file have all color scheme styles.
  3. fullcalendar.css = This CSS file is used to style Calendar on events page
  4. fancybox.css = This CSS file is used for lightbox for gallery and blog images
  5. base.css = This CSS file is used for basic common styles
  6. layout.css = This CSS file is used for responsive code creation
  7. player.css = This CSS file is used for HTML5 audio player
  8. skeleton.css = This CSS file is used for grid structure

All css rules are properly nested and commented

	
/* General Rules
================================================== */

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.
